24/// Provides hint values to the memory allocator on how long the
25/// memory is expected to be used.

29 DT_ALLOC_TEMP ///< Memory used temporarily within a function.

32/// A memory allocation function.

34// @param[in] rcAllocHint A hint to the allocator on how long the memory is expected to be in use.
35// @return A pointer to the beginning of the allocated memory block, or null if the allocation failed.
36/// @see dtAllocSetCustom

39/// A memory deallocation function.
