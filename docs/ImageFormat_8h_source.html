503 /** Converts between arbitrary formats on the CPU. Not all format conversions are supported or directly supported.
504 Formats without direct conversions will attempt to convert through RGBA first.
505
506 A conversion routine might only support source or destination padding or y inversion or none.
507 If support is needed and not available in any of the direct conversion routines, then no conversion is done.
508
509 YUV422 expects data in YUY2 format (Y, U, Y2, v). Most YUV formats require width and heights that are multiples of 2.
510
511 Returns true if a conversion was available, false if none occurred.
512
513 \deprecated
514 \sa G3D::ImageConvert
515 */
516 static bool convert(const Array<const void*>& srcBytes, int srcWidth, int srcHeight, const ImageFormat* srcFormat, int srcRowPadBits,
