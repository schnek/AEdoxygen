28/**
29\brief Easy loading and saving of XML and HTML files.
30
31The XML class is intended primarily for interchange with other
32programs. We recommend using G3D::Any to make your own human-readable
33formats because it is a more general syntax, the implementation is
34more efficient, and contains better error handling.
35
36Every XML is either a <i>VALUE</i>, or a <i>TAG</i> that contains both
37a table of its XML attributes and an array of its children. Children
38are nested tags and the strings between the nested tags.
39
40No validation is performed, and the XML must be completely legal. XML
41Entity references (e.g., the ampersand codes for greater than and less
42than) are not automatically converted.
43
44Tags with names that begin with "!" or "?" are ignored. Comment tags must
45end with "-->" e.g.,
46
47<pre>
48 <?xml version="1.0" encoding="ISO-8859-1"?>
49 <!DOCTYPE note SYSTEM "Note.dtd">
50 <!-- a comment -->
51</pre>
52
53\sa G3D::Any, http://www.grinninglizard.com/tinyxml/
54
55\htmlonly
56<pre>
57<foo key0="value0" key1="value1">
58 child0 ...
59 <child1>...</child1>
60 child2 ...
61</foo>
62</pre>
63\endhtmlonly
64*/
