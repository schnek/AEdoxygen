Parameters
msgThe encoded data to decode
msglenThe length of the encoded data (octets)
lparamThe session or system data (can be NULL)
lparamlenThe length of the lparam
modulus_bitlenThe bit length of the RSA modulus
hash_idxThe index of the hash desired
out[out] Destination of decoding
outlen[in/out] The max size and resulting size of the decoding
res[out] Result of decoding, 1==valid, 0==invalid
Returns
