Parameters
sigThe signature data
siglenThe length of the signature data (octets)
hashThe hash of the message that was signed
hashlenThe length of the hash of the message that was signed (octets)
paddingType of padding (LTC_LTC_PKCS_1_PSS or LTC_LTC_PKCS_1_V1_5)
hash_idxThe index of the desired hash
saltlenThe length of the salt used during signature
stat[out] The result of the signature comparison, 1==valid, 0==invalid
keyThe public RSA key corresponding to the key that performed the signature
Returns
CRYPT_OK on success (even if the signature is invalid)
