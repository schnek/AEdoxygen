21/** Represents numbers on [0, 1] in 8 bits as an unsigned normalized
22 0.8 fixed-point value using the same encoding scheme as OpenGL.
23
24 OpenGL specifications can be found here:
25 <www.opengl.org/registry/specs/ARB/shading_language_packing.txt>
26
27 Note that arithmetic operations may over and under-flow, just like
28 uint8 arithmetic.
29
30
31*/

57 /** Maps f to round(f * 255).*/
